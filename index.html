What I Learned About HTML Elements

For this project, I explored two HTML elements I hadn't used before: the <video> element and the <details> element (which you're seeing right now!).

The <video> element is fascinating because it provides native support for playing videos without requiring third-party plugins. I learned that it supports multiple source formats for browser compatibility, allows for custom controls, and can be styled with CSS to fit different layouts. One interesting behavior I discovered is that the poster attribute lets you display a thumbnail before the video plays.

The <details> element (which you just clicked on) creates an interactive disclosure widget that reveals additional information when toggled. It includes a <summary> element that serves as the visible heading. What surprised me was that this functionality is built directly into HTML without requiring JavaScript, making it an accessible way to organize content without cluttering the page.
